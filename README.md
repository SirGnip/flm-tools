# Installation

Follow this basic process to get a low ceremony way to get easy access to the CLI tool from anywhere:

Create a uv venv environment. This will do an "editable install" of the project in a venv. If you add the local
venv to the `PATH`, you can use the command line tool from any directory and also edit the code live.   

```
export PATH=$PATH:/dir/repo/flm-tools/.venv/Scripts/
flm --help
```
